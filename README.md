# Crystallis System v1.0 

Sistema de gestão odontológica desenvolvido em **Python** e **SQLite**. 
Focado em clínicas que buscam controle de prontuário, agenda e faturamento.

## Funcionalidades
- **Autenticação Segura:** Senhas criptografadas com SHA-256.
- **Odontograma Clínico:** Registro de procedimentos por dente (Padrão 11-48).
- **Gestão Financeira:** Relatórios de faturamento total e por paciente.
- **Controle de Acesso:** Diferentes permissões para Dentistas e Recepcionistas.

## Tecnologias
- Linguagem: Python 3.x
- Banco de Dados: SQLite3
- Arquitetura: Modularizada (src/modules)

## Como Executar
1. Clone o repositório.
2. Execute o arquivo `main.py`.
